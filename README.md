#  Système de Gestion de Magasin - C++

Application console pour la gestion d'achats en magasin.

## Fonctionnalités
- Authentification employé (5 utilisateurs)
- Catalogue de 8 articles
- Gestion de panier (ajout/retrait)
- Facture avec taxes (TPS/TVQ) et rabais aléatoire (50% chance - 25%)
